== Installation ==

1. Upload the 'wp-sales-letter' folder to the '/wp-content/plugins/' directory
2. Activate the WP Sales Letter plugin through the 'Plugins' menu in WordPress
3. Configure the plugin by going to the WPSalesLetter menu that appears in your admin menu
